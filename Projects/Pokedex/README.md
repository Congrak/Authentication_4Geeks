## Pokedex Challenge

I create this app using React + Vite

## Why am i using Vite?

- More faster for small projects
- More easier to configure
- More lighter than another development tools

## Requirements

Make sure you are using node version 7+

- Install the packge: npm install
- Run: npm run dev

- Vite
- React
- Boostrap

- CSS
